### Dependency Updates

Updating NPM

> `npm install -g npm`

Updating CLI

> `npm uninstall -g angular-cli @angular/cli`
>
> `npm chache verify`
>
> `npm install -g @angular/cli`

### Process

To create new Applicaiton Folder : `ng new app-name`

Build source code and typescript : `ng serve`

To Change Port : `ng serve --port YOURPORT`
